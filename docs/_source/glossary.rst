========
Glossary
========

.. glossary::

    ephemerality
        A software-defined resource that lasts only a short period of time.

    idempotence
        test it

    immutability
        The software concept that says we should treat a package or service as non-upgradeable
        and unchangeable.

    Mean Time To Deploy (MTTD)
        A key performance indicator for DevSecOps work.

    Virtual Private Cloud (VPC)
        A logically isolated section of the AWS Cloud where you can launch AWS resources in a 
        virtual network that you define.
